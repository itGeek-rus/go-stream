# go-stream
**GoStream** — это типобезопасная ETL-библиотека для Go, вдохновленная Python Pandas и Java Streams. Она позволяет строить потоковые пайплайны обработки данных с минимальным потреблением памяти, используя дженерики (Go 1.18+) и встроенную поддержку параллельной обработки (fan‑out/fan‑in) без ручного управления горутинами.

Особенности

- **Безопасная типизация** — все операции (`Map`, `Filter`, `GroupBy`, `Join`) параметризованы типами.
- **Потоковая обработка** — данные читаются и обрабатываются чанками, не загружая весь набор в память.
- **Параллелизм из коробки** — достаточно вызвать `.Parallel(workers)`, и пайплайн автоматически распределит нагрузку по воркерам.
- **Чистая архитектура** — легко подменять источники данных (CSV, Parquet, JSON) и приёмники, не меняя логику обработки.
- **Готовые адаптеры** — встроенная поддержка CSV и Parquet, а также слайсов для тестирования.
- **Управление контекстом** — поддержка `context.Context` для отмены и таймаутов.
